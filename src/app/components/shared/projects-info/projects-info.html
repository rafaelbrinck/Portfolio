Controle Financeiro e Orçamentos

Desenvolvido em Angular com Supabase como backend, este sistema oferece uma solução prática e responsiva para controle financeiro e emissão de orçamentos de forma profissional.

  • Login com Supabase e senhas criptografadas (bcrypt)
  • CRUD de clientes, produtos e orçamentos
  • Envio de orçamentos via WhatsApp com mensagem formatada
  • Modais reutilizáveis para feedbacks centralizados
  • Layout responsivo com Bootstrap + Bootstrap Icons
  • Proteção de rotas com JWT simulado (RouteGuard)
